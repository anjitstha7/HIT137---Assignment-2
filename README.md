## Question 1 – Encryption & Decryption (`question1.py`)

### What it does

This program reads a text file (`raw_text.txt`), encrypts its contents using a custom cipher, decrypts it back, and verifies the result.

### Encryption Rules

| Character | Rule |
|-----------|------|
| Lowercase **a–m** | Shift **forward** by `shift1 × shift2` positions |
| Lowercase **n–z** | Shift **backward** by `shift1 + shift2` positions |
| Uppercase **A–M** | Shift **backward** by `shift1` positions |
| Uppercase **N–Z** | Shift **forward** by `shift2²` positions |
| Spaces / numbers / symbols | **No change** |

All shifts wrap around the alphabet using modulo 26.

### How to Run

```bash
python3 question1.py
```

The program will:
1. Ask you to enter `shift1` and `shift2`
2. Encrypt `raw_text.txt` → saves `encrypted_text.txt`
3. Decrypt `encrypted_text.txt` → saves `decrypted_text.txt`
4. Verify that the decrypted text matches the original

### Example

```
Enter shift1: 2
Enter shift2: 3

Encryption complete!
Decryption complete!
Verification PASSED: The decrypted text matches the original!
```

## Question 2 – Expression Evaluator (`evaluator.py`)

### What it does

This program reads mathematical expressions from a text file (one per line), evaluates each one using **recursive descent parsing**, and writes the results to `output.txt`.

### Supported Features

- Basic operators: `+`, `-`, `*`, `/`
- Parentheses (nested to any depth): `(3 + (2 * 4))`
- Unary minus: `-5`, `--5`, `-(3 + 4)`
- Correct operator precedence (`*` and `/` before `+` and `-`)
- Error handling: unknown characters, division by zero, unary `+`

### How to Run

```bash
python3 evaluator.py sample_input.txt
```

This reads `sample_input.txt` and writes results to `output.txt` in the same folder.

### Output Format

Each expression produces a 4-line block in `output.txt`:

```
Input: 3 + 5
Tree: (+ 3 5)
Tokens: [NUM:3] [OP:+] [NUM:5] [END]
Result: 8

Input: 1 / 0
Tree: (/ 1 0)
Tokens: [NUM:1] [OP:/] [NUM:0] [END]
Result: ERROR
```

### Sample Results

| Expression | Tree | Result |
|------------|------|--------|
| `3 + 5` | `(+ 3 5)` | `8` |
| `2 + 3 * 4` | `(+ 2 (* 3 4))` | `14` |
| `-(3 + 4)` | `(neg (+ 3 4))` | `-7` |
| `--5` | `(neg (neg 5))` | `5` |
| `(10 - 2) * 3 + -4 / 2` | `(+ (* (- 10 2) 3) (/ (neg 4) 2))` | `22` |
| `3 @ 5` | `ERROR` | `ERROR` |
| `1 / 0` | `(/ 1 0)` | `ERROR` |
